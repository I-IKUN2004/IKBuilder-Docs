# 更新日志

## v1.3.0 (高阶交互与引擎重构更新)
* **[交互] 木斧连招双向微调：** 潜行状态下左键点击任意方块，可无缝切换“扩张”与“收缩”模式；潜行右键点击方块表面，即可直接执行边界微调，彻底告别繁琐表单。
* **[机制] 准星追踪自动预填：** 打开粘贴面板时，底层射线将动态捕获当前视线正对的远端方块坐标，并自动预填至目标定位点，实现零数字输入体验。
* **[快捷] 指南针一键盲贴：** 手持指南针右键点击任意方块，系统将绕过所有 GUI 界面，直接以该方块为基准点精准粘贴剪贴板结构，让连续建造如丝般顺滑。
* **[新增] 视距画笔与地形平滑：** 全面挂载烈焰棒专属画笔。无需框选 A/B 点，通过底层 API 追踪视线落点，即可远程瞬间完成球形雕刻与剧烈地形的平均化平滑。
* **[重构] 高级粘贴与连环堆叠：** 彻底重构剪贴板内核，现已支持输入任意绝对坐标进行跨空域精准粘贴。新增多维度连环堆叠（Stack）功能，造桥、铺路、拉围墙只需一键设定循环次数。
