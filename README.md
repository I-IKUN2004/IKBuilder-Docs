# IKBuilder (IK系列建造助手)

* **当前版本：** v1.6.0
* **开发者：** I IKUN2004
* **运行平台：** LeviLamina (LSE API)

基于 LeviLamina 引擎开发的高阶服务端建筑辅助插件。

## 核心特性

* **全异步运算防卡顿**
  * 针对大面积的填充、替换和几何体生成，底层采用 16x16x16 微切片分割技术。
* **极速物理连招 (Undo/Redo)**
  * 独创 Q 键丢弃物品极速触发防毁撤销与重做，一键清空选区，彻底告别繁琐菜单。
* **可视化粒子外框**
  * 选区和蓝图预览采用粒子实时渲染三维边界矩阵。
