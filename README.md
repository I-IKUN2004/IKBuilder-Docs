# IKBuilder (IK系列建造助手)

* **当前版本：** v1.3.0
* **开发者：** I IKUN2004
* **运行平台：** LeviLamina (LSE API)

基于 LeviLamina 引擎开发的高阶服务端建筑辅助插件。

## 核心特性

* **全异步运算防卡顿**
  * 针对大面积的填充、替换和几何体生成，底层采用 16x16x16 微切片分割技术。
* **五级防毁撤销机制 (Undo)**
  * 所有方块变动前自动捕获快照，支持连续回退 5 步。
* **可视化粒子外框**
  * 选区和蓝图预览采用粒子实时渲染边界矩阵。
