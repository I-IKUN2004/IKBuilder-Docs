📖 IKBuilder (IK系列建造助手) 官方说明文档插件名称： IKBuilder系列名称： IK系列当前版本： v1.3.0开发者： I IKUN2004运行平台： LeviLamina (LSE API) / JavaScript1. 核心特性概述IKBuilder 是一款专为 LeviLamina 引擎打造的高阶服务端建筑辅助插件。它在提供全面三维空间修改能力的同时，秉持“舒适优先”的交互设计理念，摒弃了繁琐的指令输入，全方位采用双行文本解析的 GUI 表单进行交互。全异步运算防卡顿： 针对大面积的填充（Fill）、替换（Replace）和几何体生成，底层采用 $16 \times 16 \times 16$ 的微切片分割技术配合异步回调（setTimeout），彻底告别大规模生成导致的服务端主线程卡死。五级防毁撤销机制 (Undo)： 所有的方块变动（填充、几何、粘贴、蓝图导入）前，系统会自动捕获原区域的 .mcstructure 快照，支持连续回退 5 步，保障建筑安全。可视化粒子外框： 选区和蓝图预览分别采用末地烛（End Rod）和村民高兴（Villager Happy）粒子实时渲染边界矩阵，实现“所见即所得”。高阶蓝图与剪贴板： 支持本地剪贴板的复制、精准粘贴与多向阵列堆叠，并原生对接 .mcstructure 格式文件的导入导出。2. 部署与目录结构插件启动后，会自动在服务端根目录生成以下文件结构，请确保服务端具备完整的文件读写权限：plugins/IKBuilder/ —— 根目录structures/ —— 用于存放导出的蓝图或外部导入的 .mcstructure 文件。undo/ —— 存放玩家防毁快照数据的临时缓存。auth/ —— (KVDB) 普通玩家的白名单授权数据库。license/ —— (KVDB) 服务器的机器码与卡密授权数据库。3. 安装与激活指南IKBuilder 搭载了云端硬件级授权系统（绑定机器码与目录）。将 IKBuilder.js 放入 plugins 目录并重启服务器（或 ll reload）。进入服务器控制台或在游戏内拥有 OP 权限，输入指令 /ikb <卡密>（例如 IK30-xxx）连接云端验证节点。激活成功后，后台会自动写入过期时间并锁定当前机器哈希。此后输入 /ikb 即可打开主菜单。4. 详细功能模块说明A. 选区与坐标系统这是所有区域操作的基础。快捷选区工具 (木斧)： * 左键点击 (或破坏方块)： 设定 A 点（起点）。右键点击方块： 设定 B 点（终点）。微调边界 (高阶技巧)： 玩家手持木斧潜行并右键方块任意面，选区边界会自动向该方向扩张 1 格。再次潜行破坏方块可将模式切换为“收缩模式”。绝对坐标校准： 在菜单中可以手动输入精确的 XYZ 绝对坐标作为 A/B 点。B. 区域建筑与修改对选定的 A/B 矩阵执行覆盖操作：批量填充 (Fill) & 精准替换 (Replace)： 自动识别手持方块的 ID 作为默认值，支持 TileData 输入。几何体生成 (Shapes)： 以玩家当前脚下坐标为中心生成。支持形状： 完美球体、圆柱体、正方/长方体、圆锥体、半球/圆顶。空心模式： 支持真·空心化（仅保留外壳，内部自动挖空）。时光倒流 (Undo)： 独立菜单面板操作，按后入先出（LIFO）顺序回档最近的操作，伴随精准的掉落物压制（防掉落卡服）。C. 结构蓝图与剪贴板 (Clipboard & Structures)复制与粘贴：复制时，系统会自动记录建筑相对于玩家脚下的偏移量（Offset）。粘贴或堆叠时，菜单会自动获取玩家准星正在凝视的方块坐标作为预填坐标点，极大提升放置效率。多向阵列堆叠 (Stack)： 支持向上、下、东南西北 6 个方向设定次数进行连环堆叠。蓝图导入与预览：加载 structures 文件夹内的蓝图后，不会立刻放置，而是生成绿色粒子外框。支持中心点旋转（0/90/180/270度）以及 X/Z 轴对称镜像翻转。D. 画笔与地形工具 (Brushes)绑定物品为 烈焰棒 (Blaze Rod)。右键点击方块或悬空凝视即可触发。球形雕刻画笔： 远程在视线落点快速生成设定半径的球体。柔和地形平滑 (Smooth)： 自动扫描目标区域内的方块最高点，计算平均高度（AvgY），将凹陷处填平，将凸起处削去，用于快速修整自然地形。E. 权限控制 (Auth)默认仅 OP（权限等级 >= 1）可以使用该插件。OP 可在“权限与成员管理”菜单中，通过游戏列表或精确输入 XUID 的方式，将普通玩家加入构建者白名单。5. 指令与权限字典为了保证 LSE 引擎生命周期管理的稳定性，核心指令注册逻辑已严格置于脚本最外层同步环境，避免了重载时的内存泄漏（get on empty Global 崩溃）。指令参数权限要求功能描述/ikb[cdk] (可选)任何人 (PermType.Any)打开 IKBuilder 创世神主界面。未激活时若带有 CDK 参数则发起云端验证。注意： 普通玩家即使输入 /ikb，若其 XUID 不在内置的 KVDB 白名单中，也会被系统拦截并提示权限不足。6. 给服主的防坑与注意事项关于撤销机制： 撤销操作底层依赖 mc.getStructure。如果玩家的选区过大，或者选区边缘跨越了尚未被加载的区块 (Chunk)，快照将捕获失败。请确保进行大规模操作前，目标区域已在视野内加载。掉落物压制： 插件在执行粘贴或撤销时，会短暂关闭游戏规则 dotiledrops 并清理目标区域内的掉落物，防止覆盖容器类方块时瞬间爆出海量物品导致服务器崩溃。Litematica (投影) 转换： 插件内置了提示，建议使用网页端 Bloxelizer 将 Java 版的 .litematic 投影文件安全转换为基岩版的 .mcstructure，随后放入 structures 文件夹即可直接调用。
