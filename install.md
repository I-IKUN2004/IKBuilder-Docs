# 安装与激活

IKBuilder 搭载了云端硬件级授权系统（绑定服务器机器码与安装目录），普通玩家需通过白名单系统获取权限。

## 核心安装步骤

1. 将下载好的 `IKBuilder.js` 放入 LeviLamina 的 `plugins` 目录。
2. 重启服务器，或在控制台执行 `ll reload`。
3. 首次运行后，插件会自动在 `plugins/IKBuilder` 目录下生成 `auth`、`license`、`structures` 等系统文件夹。

## 授权与激活 (CDK)

插件运行必须激活。如果未激活，任何玩家输入指令都会被拦截。

* **获取授权：** 请联系开发者 (I IKUN2004) 获取您的专属授权卡密 (CDK)。
* **执行激活：** 在服务器后台控制台，或在游戏内使用 OP 账号，输入以下指令进行验证：
  `/ikb <您的卡密>`
  *(例如：`/ikb IK30-ABCDEFG`)*
* **激活成功：** 后台会显示绿色的授权通过日志，并自动记录机器特征与到期时间（支持 30天/90天/365天/永久）。

## 权限与白名单配置

出于安全考虑，默认情况下 **仅拥有 OP 权限（权限等级 >= 1）的管理员** 可以打开创世神面板。
如果您想让特定的普通玩家（建筑师）使用：
1. OP 玩家输入 `/ikb` 打开面板。
2. 进入 **[5] 权限与成员管理**。
3. 您可以从在线列表中快速添加，或手动输入玩家完整的 XUID 添加到构建者白名单中。
