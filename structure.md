# 蓝图与剪贴板

支持高阶的内部相对结构复制，以及与外部系统的 `.mcstructure` 原生对接。

## 剪贴板 (复制、粘贴与堆叠)
* **复制结构：** 复制选区时，系统会自动记录该结构与您**脚下坐标的相对偏移量**。
* **高阶粘贴：** 粘贴面板会自动捕获您**准星正对的远端方块**作为预填坐标，实现类似《原神》尘歌壶般的精准放置。
* **线性连环堆叠：** 在粘贴面板可切换至 Stack 模式，支持向 上/下/东/南/西/北 六个方向，指定次数进行完美无缝拼接铺设，造桥铺路神器。

## 导入与导出
* **导出：** 在面板内一键将当前选区序列化并保存至服务器后台的 `plugins/IKBuilder/structures` 目录。
* **导入外部蓝图：**
  将 `.mcstructure` 文件放入上述目录后，在菜单中选择加载。
  系统**不会直接放置**，而是生成绿色粒子外框进行预览。您可以确认并微调坐标后，再执行最终放置。
* **旋转与翻转：** 支持 0/90/180/270 度旋转及 X/Z 轴镜像对称。

## 外部格式转换
如果是 Java 版投影文件 (.litematic)，强烈建议通过 `Bloxelizer` 在线网站免费转换为 `.mcstructure` 后导入，支持度完美。
