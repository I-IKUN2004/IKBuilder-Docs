# 指令与权限

为了贯彻“舒适交互”的理念，IKBuilder 彻底摒弃了传统创世神繁琐的 // 文本指令系统。所有的功能均统一在一个主入口指令下，并通过全 GUI 界面完成。

## 核心指令

| 指令 | 参数说明 | 默认权限要求 | 功能描述 |
| :--- | :--- | :--- | :--- |
| `/ikb` | `[cdk]` (可选) | 任何人可执行* | 打开 IKBuilder 创世神主界面，或进行 CDK 激活。 |

*> **注：** 虽然系统层面注册为任何人均可执行，但插件内部会进行二次鉴权。未在白名单中的普通玩家执行会被直接提示“权限不足”并拒绝打开 GUI。*

## 为什么没有其他子指令？
在 LeviLamina (LSE) 插件开发机制下，为了保障极高的稳定性与防止指令重载残留导致内存泄露，本插件采用底层同步挂载单入口设计。一切复杂的参数（如方块ID、数据值、坐标点）均交由 GUI 表单中的输入框和滑动条来完成。
